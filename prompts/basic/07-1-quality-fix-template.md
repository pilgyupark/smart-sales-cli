# 07-1. 품질 점검 후 개별 수정 템플릿

```text
앞서 보고한 품질 문제 중 {{수정할 항목}}만 수정해줘.

[작업 branch]
- ai/quality-fix-{{주제}}

[변경 허용 범위]
- {{파일 1}}
- {{파일 2}}

[완료 조건]
- 선택한 문제만 수정
- 관련 unittest 보완
- 전체 unittest 실행

[금지 사항]
- 다른 품질 문제는 아직 수정하지 마.
- 허용 범위 밖 파일은 수정하지 마.
- 관련 없는 리팩터링을 하지 마.

먼저 현재 branch, 최소 수정 계획, 수정 대상 파일, 테스트 방법을 설명하고 승인을 기다려줘.
```
