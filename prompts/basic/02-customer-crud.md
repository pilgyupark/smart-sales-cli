# 02. 고객사 관리 구현

```text
Smart Sales CLI에 고객사 관리 기능을 추가해줘.

[작업 branch]
- ai/customer-crud

[변경 허용 범위]
- customer_service.py
- validators.py
- app.py
- tests/test_customer_service.py

[완료 조건]
- 고객사 등록
- 고객사 목록 조회
- 고객사 상세 조회
- 고객사 수정
- 고객사 삭제
- customer_id 중복 등록 차단
- 이메일 형식 검증은 validators.py의 공개 함수를 사용
- 관련 unittest 추가

[금지 사항]
- storage.py 수정이 필요하면 먼저 이유를 설명하고 추가 승인을 기다려줘.
- 기존 JSON 필드명을 변경하지 마.
- 검색 기능은 아직 추가하지 마.
- 불필요한 리팩터링을 하지 마.

먼저 현재 branch, 변경 계획, 수정 대상 파일, 추가할 테스트를 설명하고 승인을 기다려줘.
```
