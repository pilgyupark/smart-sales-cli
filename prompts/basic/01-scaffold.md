# 01. 프로젝트 뼈대 생성

```text
REQUIREMENTS.md와 .clinerules의 규칙을 지켜 Smart Sales CLI의 초기 뼈대를 만들어줘.

[작업 branch]
- ai/scaffold

[변경 허용 범위]
- storage.py
- validators.py
- app.py
- 필요한 경우 data 폴더의 JSON 파일 초기화

[완료 조건]
- data 폴더의 JSON 파일을 읽고 저장할 수 있어야 해.
- app.py에는 실행 가능한 기본 메뉴와 종료 기능만 구현해줘.
- Python 표준 라이브러리만 사용해줘.

[금지 사항]
- 허용 범위 밖의 파일은 생성하거나 수정하지 마.
- 고객사 CRUD, 검색, 영업일지, 결재, 요약 기능은 아직 구현하지 마.
- 외부 패키지를 설치하지 마.

먼저 현재 branch, 작업 계획, 생성하거나 수정할 파일, 실행 방법을 설명하고 승인을 기다려줘.
```
